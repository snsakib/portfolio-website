Overview

"Agency" is a corporate website landing page. It is fully responsive in design. This webpage is developed as a personal project of mine to refurbish my front-end web development skills.

Technologies Implemented

  • Angular
  • Angular Material
  • HTML 5
  • CSS 3
  • Sass
  • TypeScript
  • Git
  • Zeplin
  • Photoshop

Why I Did This Project?

After grasping the fundamentals of front-end web development, I decided to sharpen my skills by working on some projects. Therefore, I came up with the idea to develop this responsive webpage.

Challenges

To make webpage responsive
The design template I’d chosen for this project was designed with big screens in mind. My challenge was to make this design responsive so that the users can have a smooth experience in all devices irrespective of the screen sizes.
To optimize initial loading time
Another challenge I encountered, was with the images. There are in total of 18 images in this webpage. Some of them were very large in file size (~2 MB). It drastically affected the initial loading time of the webpage (5-7s according to Lighthouse audit).

Development Process

Phase 1: Planning
Before diving into coding, I like to think for a while to outline a strategy. I usually don’t spend too much time planning. I seek to be as AGILE as possible.
During this phase, I basically decided which technologies & tools I would use in this project.
Phase 2: Coding (Resolving first challenge)
As I mentioned earlier, my span first challenge was to make this webpage responsive. I applied “CSS Flexbox” & “Media Queries” to accomplish that. You can visit the GitHub repo for this project & explore the codebase.
Phase 3: Refactoring (Resolving second challenge)
After developing the MVP (Minimum Viable Product) of my webpage, I turned my attention into small details.
To resolve my second challenge, I optimized the images (using Squoosh) & was able to reduce the file sizes by 30% on average. I implemented responsive image techniques to render images. As a result, the initial loading time reduced to 0.9-1s.
