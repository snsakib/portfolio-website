Overview

"Agency" is a corporate website landing page. It is fully responsive in design. This website is developed as a personal project of mine to refurbish my front-end web development skills.

# Technologies & Tools Used

Angular, Angular Material, HTML, CSS, Sass, JavaScript ES6, TypeScript, Git, GitHub, Zeplin, PhotoShop

Why Did I Do This Project?

After grasping the fundamentals of front-end web development, I decided to sharpen my skills by working on some projects. Therefore, I came up with the idea to develop this responsive website.

Challenges

The design template I’ve chosen for this project was designed with big screens (i.e. laptops, desktops) in mind.

  • My first challenge was to make this design responsive so that the users can have a smooth experience in all devices irrespective of the screen sizes.
  • Another challenge I encountered, was with the images. There are in total of 18 images in this website. Some of them were very large in file size (2 MB+). It drastically affected the initial loading time of the website (5-7s according to Lighthouse audit).

Development Process

Phase 1: Planning

Before diving into coding, I like to think for a while to outline a strategy. I usually don’t spend too much time planning. I seek to be as AGILE as possible.
During this phase, I basically decided which technologies & tools I would use in this project.

Phase 2: Coding

As I mentioned earlier, my first challenge was to make this website responsive. I applied “CSS Flexbox” & “Media Queries” to accomplish that.
You can visit the GitHub repo for this project & explore the codebase.

Phase 3: Refactoring

After developing the MVP (Minimum Viable Product) of my website, I turned my attention into small details.
To resolve my second challenge, I optimized the images & was able to reduce the file sizes by 30% on average. I applied the “srcset” attribute to dynamically render images depending on the screen size (small images on small screens & large images on large screens). As a result, the initial loading time reduced to 0.9-1s.

My Top Takeaways

Angular Ecosystem

This project tested my knowledge on overall Angular ecosystem. It helped me to explore the whole process of “Creating To Deploying” an Angular app.

Responsive Images

After working on this project, I’m now capable of optimizing images for achieving a better performance.

Persistency

To be honest, there were times I couldn’t figure out what went wrong. At points, I felt devasted. But I never thought of quitting. This lesson of grit will be very useful not only for achieving my career goals but also in my personal development.
